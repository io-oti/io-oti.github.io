# VitePress Theme

This is the custom theme for VitePress. It is a simple and clean design that focuses on content and readability.

```shell
docs/.vitepress
├── configs
│   ├── themeConfig
│   │   ├── index.js              # Theme config
│   │   ├── search.js             # Search config
│   │   ├── sidebar.js            # Sidebar config
│   │   └── socialLinks.js        # Social links config
│   ├── index.js
│   ├── markdown.js               # Markdown config
│   └── vite.js                   # Vite config
├── theme
│   ├── components
│   │   ├── Collapse              # Collapse component
│   │   ├── Comments              # Comments component
│   │   ├── Icons                 # Icons component
│   │   └── Layout                # Layout component
│   ├── hooks
│   │   ├── useAppearance.js      # Change appearance animate
│   │   ├── useCrawler.js         # Dark appearance background animate
│   │   └── useFirefly.js         # Light appearance background animate
│   ├── pages
│   │   ├── Archives              # Archives page
│   │   ├── Categories            # Categories page
│   │   └── Home                  # Home page
│   ├── styles
│   │   ├── catppuccin-latte.css  # Catppuccin Latte theme styles
│   │   ├── catppuccin-mocha.css  # Catppuccin Mocha theme styles
│   │   ├── component.css         # Component styles
│   │   └── theme.css             # Theme styles
│   ├── index.js                  # Theme entry file
│   └── posts.data.js             # Posts data
├── config.mjs                    # VitePress config file
└── README.md
```
